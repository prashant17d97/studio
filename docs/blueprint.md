# **App Name**: ProInvoice

## Core Features:

- Invoice Data Input: Allow users to input invoice details, including customer information, line items, taxes, and discounts.
- PDF Preview: Provide a real-time preview of the generated PDF invoice to ensure accuracy before downloading.
- PDF Generation with Pagination: Automatically generate multi-page A4 PDF invoices with intelligent pagination to handle large amounts of data.
- Invoice Download: Enable users to download the generated invoice in PDF format.
- Invoice Management: Allow users to manage (list, view, delete) previously created invoices.

## Style Guidelines:

- Primary color: Deep Blue (#003399) for a professional and trustworthy feel, closely replicating the original design.
- Background color: Light Grayish-Blue (#E6EEF9) providing a clean and minimalist backdrop, inspired by the reference.
- Accent color: Slightly lighter blue (#3366CC) to highlight interactive elements and key information, maintaining visual consistency.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable, and professional look, matching the sleek design of the original.
- Use simple, outlined icons in a light blue tone (#3366CC) for visual clarity and a minimalist aesthetic, mirroring the example's understated icons.
- Maintain a clean, well-spaced layout with clear sections for invoice details, mirroring the organized structure of the DeepCarve invoice design.
- Incorporate subtle animations on button hover states to provide user feedback and enhance the overall user experience.
